## Get Started
1. Clone Repository
```bash
git clone https://github.com/dimmasyusuf/noteable.git
```

2. Open Directory
```bash
cd noteable
```

3. Start Server
```bash
npm run start
```

## Build for Production

- Build
```bash
npm run build
```
